Read from Save File Generate an optimal display from a save file

By uploading your save file, this application will check your inventory and chests to find the best possible grange display for the Stardew Valley Fair.

By default, your save file will contain two versions, the SaveGameInfo file and one with your farmers 9 digit ID, e.g. Farmer_224805264. You should use the file with the 9 digit ID as the other one doesn't contain the information required for the app to work.

The Default save file locations are as follows:

  • Windows: %AppData%\StardewValley\Saves\
  • Mac OSX: ~/.config/StardewValley/Saves/
  • Linux: ~/.config/StardewValley/Saves/
